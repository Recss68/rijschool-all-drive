De officiële website van **Rijschool All Drive**, gebouwd met een moderne en schaalbare webstack.  
De focus ligt op **performance**, **SEO** en **gebruiksvriendelijkheid**, met ruimte voor toekomstige uitbreidingen zoals online inschrijvingen en leerlingaccounts.

---

## ✨ Features

- ⚡ Supersnel dankzij SvelteKit
- 🔍 SEO-vriendelijk (SSR & SSG)
- 📱 Volledig responsive (mobile-first)
- 📝 Online inschrijfformulieren
- 🗄️ Backend & database via Supabase
- 🔐 Veilige data-afhandeling met Row Level Security
- 🚀 Automatische deploys via Vercel

---

## 🧱 Tech Stack

**Frontend**

- Svelte
- SvelteKit
- JavaScript (ESNext)

**Backend / Services**

- Supabase (Database, Auth, Storage)

**Deployment**

- Vercel
